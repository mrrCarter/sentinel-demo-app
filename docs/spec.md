# Spec

sentinel-demo-app

## Goal
A minimal Express.js REST API with JWT-only authentication, email verification via Resend, and AIdenID E2E testing. Built to demonstrate the full SentinelLayer CLI workflow including Omar Gate security review on every PR.

intermediate

## Preferred provider
anthropic

## Project type
greenfield

## Suggested stack
Node.js, TypeScript, PostgreSQL

## Key features
1. Implement the core workflow end-to-end.
2. Add observability and hardening.
3. Add tests and docs.
